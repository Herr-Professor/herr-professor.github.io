Software engineer building ML infrastructure, deterministic evaluation systems, and performance-critical backends.

I work across Python and C++ on reproducible evaluation harnesses, verifiable data pipelines, backend systems, and systems tooling. Recent projects include RL-style code evaluation environments, high-throughput data pipelines with auditability, compression tooling, and custom memory-management experiments.

## Focus areas

- Reproducible data and benchmark pipelines
- Backend systems in Python and C++
- Verification, observability, and performance engineering
